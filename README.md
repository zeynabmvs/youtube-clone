## Getting Started
1. Clone the repository:
```
git clone https://zeynabmvs.github.io/youtube-clone.git
```

2. Navigate to the project directory:
```
cd youtube-clone
```

3. Install dependencies:
```
npm install
```

4. Start the development server:
```
npm run dev
```
